About DVFA

Damn Vulnerable Flask App (DVFA) is a Flask based web application that is damn vulnerable. Its main goal is to provide a platform to application security enthusiasts/professionals or students to test their penetration testing skills in a legal environment. This application covers all the common vulnerabilities found in web applications (following OWASP top 10) and contains several challenges that the user can try. This application also contains a section where a user can read various articles on Flask application security. This project is developed and maintained by @brownhat57 (twitter) . DVFA is free and open source and can be downloaded from here.
